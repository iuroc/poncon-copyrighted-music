## 状态码

| 状态码 | 说明                 |
| ------ | -------------------- |
| 200    | 成功                 |
| 900    | 参数缺失             |
| 901    | 暂无查询结果         |
| 902    | 服务器错误           |
| 903    | 数据库出错           |
| 904    | 用户名或邮箱已被注册 |
| 905    | 验证码错误           |
|906|验证码过期|
